The files in this folder contains the QC analysis for bam files
converage_calc.sh: extract the coverage for each CpG based on the input bed and bam files
mbias.sh: generate mbias plot for each bam files
BASIC_MAV.R: comparing results BASiCS and MAV calculation
pesudo_combination.R: simulation of cell mixture and linear assumption deconvolution
percent_overlap_dMML_dNME: examine the percentage of regions overlapping tissue-specific GO terms
subSasmple_bam.sh: subsample bam files for pesudo_combination.R analysis
